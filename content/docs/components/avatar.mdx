## Usage

```tsx

  Avatar,
  AvatarFallback,
  AvatarImage
} from "@/components/ui/avatar"
```

```tsx

  <AvatarImage src="https://github.com/shadcn.png" alt="Shadcn avatar" />
  <AvatarFallback>CN</AvatarFallback>
</Avatar>
```
